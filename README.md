# workenv

Environment configs and executable scripts

## Directory Tree

```
workenv/
├─ dist/                         Distribution-ready configs per stack
│  ├─ ai-agent/                  AI agent tooling configs
│  ├─ application/               Shared application settings
│  ├─ bin/                       Executables
│  ├─ fish/                      fish-shell setup
│  ├─ git/                       Git templates and configs
│  ├─ homebrew/                  Homebrew tap definitions
│  ├─ misc/                      Extra configs
│  ├─ neovim/                    Neovim setup
│  ├─ scripts/                   Utility scripts
│  ├─ tmux/                      tmux profiles
│  └─ zsh/                       zsh-shell setup
└─ executables/                  Sources for `dist/bin` executables
```
